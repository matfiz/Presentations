!SLIDE
# Redis, c'est quoi ? #

* Base de données NoSQL
* Type clé/valeur
* Mémoire, avec persistance possible
* Memcached++
* Actuellement en version 2.0
* Licence BSD
* Par Salvatore Sanfilippo @antirez


!SLIDE
# Pour quoi faire ? #
## Plein de choses ##

* Cache
* Files d'attentes
* PubSub
* Statistiques
* Liste d'amis
* ...


!SLIDE
# Les forces de Redis #

* Très rapide
* Simple à installer et utiliser
* Des opérations avancées selon les types
* Persistance
* Nombreux bindings


!SLIDE
# Installation #

1. Télécharger redis
2. `make`
3. Utiliser `redis-server` et `redis-cli`
